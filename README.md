# ÉCLAT SKIN - Luxury Dynamic Storefront

A high-fidelity, Aesop-inspired single-page storefront built with React (Vite), Node.js (Express), and SQLite.

## Features
- **21 Premium Products**: Categorized into Skin Care, Body & Hand, and Kits & Travel.
- **Dynamic Search**: Real-time product discovery engine with backend filtering.
- **Interactive Tracking**: Functional order status simulation with animated feedback.
- **Smart Checkout**: LocalStorage-based address persistence and multi-stage checkout lifecycle.
- **Full Auth**: JWT-protected login and signup with personalized "Cabinet" access.
- **Contact API**: Functional enquiry form with database persistence for customer messages.

## Project Structure
- `/storefront`: React frontend (Vite, Framer Motion, Lucide Icons).
- `/backend`: Node.js Express server with SQLite3 database integration.

## Setup Instructions

### 1. Start the Backend
```bash
cd backend
npm install
npm start
```
*The server runs on http://localhost:5000*

### 2. Start the Frontend
```bash
cd storefront
npm install
npm run dev
```
*The storefront launches on http://localhost:5173*
